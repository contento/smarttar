# Introduction
SmartTar is a real-time telephone tariff management system for public telephone booth centers. It monitors active calls, calculates charges according to configurable tariff schedules, prints itemized receipts with tax, and maintains a complete transaction history.

This guide covers the day-to-day operation of SmartTar for telephone center operators and supervisors. For technical configuration details, system architecture, and hardware installation, refer to the *SmartTar Reference Manual*.

## Conventions Used in This Guide

- Menu paths are written as **Menu \> Item** (e.g., **Configuración \> Fecha y Hora**).
- Key combinations use the format **Alt+F4**, **F1**, **Enter**.
- Button labels in dialogs appear in **bold**.
- Field names in dialogs appear in *italics*.
- Important notes appear as: **Note:** text.

## Basic Navigation

SmartTar uses the Zinc graphical user interface. All windows, menus, and dialogs can be operated with either the keyboard or a mouse.

- **Menu bar.** The menu bar at the top of the screen contains the main menus. Press the underlined letter while holding **Alt** to open a menu, or click the menu title with the mouse.
- **Toolbar.** A row of icon buttons provides quick access to the most common actions.
- **Dialogs.** Use **Tab** / **Shift+Tab** to move between fields. Press **Enter** to accept (**Aceptar**) or **Esc** to cancel (**Cancelar**).
- **Help.** Press **F1** at any time for context-sensitive help for the current window.
- **Exit.** Press **Alt+F4** or close the main window. A confirmation dialog will appear.
