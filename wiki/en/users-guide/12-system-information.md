# System Information
The Information menu (**Información**) provides read-only views of the current system configuration, active cluster count, and version details.

## Información del Sistema

Opens a dialog showing:

- Number of active/configured booths (detected vs. configured clusters).
- Answer signal method in use.
- Printer form type and port.
- Current timing parameters (T_TALK, T_DIAL, T_LOCK, T_ANSWER).
- Rounding value (M_ROUND).
- Digit counts per call type.

**Path:** Información \> Información del Sistema

## Acerca de (About)

Displays the application version, serial number, and copyright notice. Also accessible via **Ayuda \> Acerca de**.
