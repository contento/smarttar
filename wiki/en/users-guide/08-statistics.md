# Statistics
SmartTar maintains running statistics across multiple time horizons, accessible from the Print menu via the Sales Accumulator dialog.

| Period | Data available                                               |
|--------|--------------------------------------------------------------|
| Day    | Calls, revenue, and tax totals for the current calendar day. |
| Week   | Rolling 7-day totals.                                        |
| Month  | Current calendar month totals.                               |
| Year   | Current calendar year totals.                                |

Statistics are automatically managed by the system. Daily statistics reset when the date changes. The statistics snapshot file (RX.STA) is updated whenever the database is flushed. No manual reset is needed for normal operation.
