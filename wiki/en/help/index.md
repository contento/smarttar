# In-app Help

The compiled in-app help (`help.dat`) is **Spanish only** — that is the language
SmartTar ships in, and the Zinc help anchors (`H_*`) resolve to the Spanish
topics in [[es/ayuda/index|es/ayuda]].

This folder is reserved for documentation-only English translations of the help
topics. It is **not** compiled into the application (see `manifest.json` →
`help.note`). Populate it only if an English build is ever required.
