Esta instalación leerá la información de configuración anterior del sistema y permitirá la configuración de los parámetros más generales; por favor solamente cambie lo necesario para configurar su sistema acorde a la localidad en que se encuentra prestando el servicio el sistema SmartTar.
Los parámetros son:
- País.
En este se programará el País en que este prestando el servicio el sistema SmartTar.
- Símbolo Monetario.
En este se programará el Símbolo Monetario del País en que este prestando el servicio el sistema SmartTar.
Ejemplo:
País: Colombia
Símbolo Monetario: $ (pesos).
- Impuesto.
En este se programará el nombre el impuesto del país en que esta prestando el servicio el sistema SmartTar.
Ejemplo:
País: Colombia
Nombre del Impuesto: IVA (Impuesto al Valor Agregado.
- Porcentaje del Impuesto.
En este se digitará el valor del porcentaje establecido para el impuesto.
- Representante.
Este se podrá seleccionar el representante que esta a cargo del sistema en esa localidad.
Ejemplo:
País: Colombia
Representante: MicroDiseño Ltda.
- Acceso.
Ente se podrá seleccionar el número de acceso para generar las llamadas internacionales y los cuales pueden ser el número 9 nueve o el 0 cero.
Ejemplo:
País: Colombia.
Acceso: 9.
- Ciudad.
En esta se digitará el nombre de la localidad en la cual esta prestando el servicio el sistema SmartTar.
Ejemplo:
Nombre de la localidad: Piedecuesta (Sdr).
- Usuario.
En este se digitará el nombre de la empresa que adquirió el sistema.
Ejemplo:
Usuario: Telecom.
- Identificación del usuario.
En este se podrá digitar el número de identificación del usuario.
Ejemplo:
Localidad: Piedecuesta (Sdr)
Identificación del usuario: NIT 899999023-9.
- Formatos de impresión.
En esta se podrá seleccionar el tipo de formato de impresión que se utilizará para los recibos de acuerdo al tipo de impresora utilizada y dependiendo del tipo de papel o la cantidad de rollos que utiliza, y a la cantidad de caracteres que maneja.
Ejemplo:
impresora LX, FX, LQ: 80 col. doble recibo
- Crear PH_INFO.DAT
El Sistema SmartTar en las versiones 2.0 y posteriores tienen una forma diferente de manejar la información, para lograr así una mayor eficiencia y tener la certeza de que, los archivos que contienen la información de la configuración del sistema y los arboles de numeración solamente puedan ser modificados por personal autorizado. Al utilizar esta opción el sistema lo que hace es tomar la información de los archivos de discado directo nacional e internacional que están trabajando en el momento en la versión anterior y que supuestamente han ido siendo organizados con el tiempo acorde a las características de la localidad y son convertidos al formato que maneja el archivo PH_INF.DAT (datos de información telefónica).que es el que trabajara directamente con el sistema.

Procedimiento:
Presione [TAB] para cambiarce entre las diferentes opciones y para modificar la opción, digite los datos deseados. En las opciones que tienen datos predeterminados presione [ENTER] y por medio de las teclas del cursor seleccione el parámetro acorde a su necesidad, luego presione nuevamente [ENTER]. Si desea la ejecución de la opción Crear PH_INFO.DAT presione [ENTER] para marcar el cuadro con X y sino es necesario utilizarla solamente presione [TAB] y pasé a Continuar luego presione [ENTER].
Para cerrar esta caja de diálogo y salir presione en cualquier momento [F3].
