Estos parámetros permiten la programación de los valores de los tiempos de operación para el manejo de las llamadas, los parámetros son:

De Gracia.
Es el tiempo que da el sistema después haber señal de contestación, para empezar a tarificar, esto le permitirá identificar a usuario si la comunicación es falsa ó equivocada, teniendo así tiempo para colgar el teléfono antes de que el sistema empiece a tarificar, para que no genere recibo.
Este tiempo se puede programar desde uno (1) hasta veinte (20) segundos, por defecto se utilizará en uno (1) segundos.

Para Marcación.
Es el tiempo que dispondrá el usuario para realizar la marcación completa. Al terminar este tiempo, si no se ha completado la marcación, el sistema cortará el proceso de la llamada y el usuario tendrá que intentar de nuevo la llamada.
Este tiempo se puede programar desde diez (10) hasta sesenta (60) segundos , por defecto se utilizará de quince (15) segundos.

De Cuelgue.
Es tiempo que se demorara el sistema en habilitar de nuevo la línea telefónica de una cabina después de haber detectado un cuelgue igual o mayor a doscientos (200) milisegundos; permitiendo así evitar que los cuelgues muy rápidos no los detecte el sistema.
Este tiempo se puede programar desde doscientos (200) hasta dos mil (2000) milisegundos, por defecto se utilizará en ochocientos (800) milisegundos
Para Contestación.
Es el tiempo de espera que dará el sistema para que una vez terminada la marcación completa, se reciba la señal de contestación, de lo contrario el sistema cortará el proceso de la llamada y el usuario tendrá que intentar de nuevo la llamada.
Este tiempo se puede programar desde quince (15) hasta setenta (70) segundos , por defecto se utilizará de veinte (20) segundos.

De Inicio de Comunicación.
Es el tiempo programado cuando el sistema para empezar a tarificar es Por Tiempo. En este caso es necesario programar también el tiempo Para Contestación, mayor que el De Inicio de Contestación con el fin de que no se corte esta.
Este tiempo se puede programar desde quince (15) hasta sesenta (60) segundos , por defecto se utilizará de veinticinco (25) segundos.

Procedimiento:
En el momento de requerir la utilización de este servicio ubíquese en la Barra de Menús presionando [ALT], y por medio delas teclas del cursor seleccione el menú Configuración, luego presione [ENTER] y el sistema colocará en pantalla una caja de diálogo descolgable, , en la cual aparecerá la información de todos los servicios prestados en este menú. Por medio de las teclas del cursor seleccione el servicio que desee utilizar; si no necesita ningún servicio, ó entró en el menú por curiosidad, presione [F3] y volverá a la Barra de Herramientas, o presione [ENTER] y el sistema colocará en pantalla una caja de diálogo en la cual aparecerá la información de todos los tiempos de operación, por medio de las teclas de cursor seleccione el tiempo a programar y digite el valor, entre rango determinado para dicha opción. Presione [TAB] y seleccione el botón de Aceptar, presione [ENTER] para ejecutar la acción ó presione [TAB] y seleccione el botón de Cancelar y presione [ENTER] para volver a la Barra de Herramientas.

Para cerrar esta caja de diálogo y volver a la caja de diálogo en la cual se estaba trabajando presione en cualquier momento [F3].
