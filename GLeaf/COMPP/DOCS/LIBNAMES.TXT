With version 3.0 of the Comm++ Library, Greenleaf is shipping multiple
compilers on a single distribution package.  Because of this we
have created a new library naming convention.

All Comm++ libraries start with a CP prefix.  Additional letters will
be appended to describe different qualities.

    F   - for Flat
    W   - for Windows
    C   - for Console
    X   - Borland PowerPack
    P   - Phar Lap TNT DOS Extender
    R   - DOS/4G DOS Extender
    O   - OS/2
    300 - Windows DLL
    30  - Windows NT DLL

Examples
    Large DOS       - CPL.LIB
    Large Windows   - CPLW.LIB
    WIN 16 DLL      - CP300.LIB
    WIN 32 Console  - CPFC.LIB
    OS/2 Flat Model - CPFO.LIB
