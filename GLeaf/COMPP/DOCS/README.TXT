The following documentation files are included with this release of
Greenleaf Comm++:


BUILD.TXT       : Information on using BUILD to update or recompile your
                  libraries.

DEMOS.TXT       : Description of the demos provided in this library.

DOS32.TXT       : A little information on the use of DOS Extenders with
                  the Comm++ Library.

FEATURE.TXT     : A listing of new features.

GFC-FAQ.TXT     : Trouble shooting -- Frequently asked questions about
                  Comm++.

IDE.TXT         : Information on using Comm++ in an Integrated Development
                  Environment.

INSTALL.TXT     : Suggestions on Comm++ installation.

LIBNAMES.TXT    : New library names for Comm++ Library 3.0.

README.TXT      : This file.  An overview of what the other files are.

XREF.TXT        : A cross listing reference.  It contains cross listing
                  of functions to source files and example programs.

These files will be installed in your DOCS directory if you select that
option during installation.
