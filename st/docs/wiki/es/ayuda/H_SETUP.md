El Sistema SmartTar en las versiones 2.0 y posteriores tienen una forma diferente de manejar la información, para lograr así una mayor eficiencia y tener la certeza de que, los archivos que contienen la información de la configuración del sistema y los arboles de numeración solamente puedan ser modificados por personal autorizado. Los archivos tienen dos funciones:
1. Entorno para el usuario en el cual se pueden observar y modificar los archivos.
2. Estos mismos archivos convertidos a un formato diferente para así trabajar con el sistema.
El proceso consiste en que el usuario modifica todo lo necesario en los archivos de entorno de usuario. Pero para que el sistema los asuma, es necesario ejecutar el archivo SETUP.EXE el cual permite convertir los archivos de entorno del usuario en archivos de entorno del sistema y viceversa. Ósea los archivos que están en el momento trabajando en el sistema convertirlos a entorno del usuario y así modificar lo que esta actualmente trabajando.
Entorno usuario. Entorno Sistema.

Solamente utilícelo cuando se determine que tiene problemas o alteraciones con los acumulados; este le hará una nueva sumatoria del los acumulados.

Procedimiento:
Presione [TAB] para cambiarce entre las diferentes opciones y para ejecutar la opción presione [ENTER].
