En este se puede seleccionar, el puerto que va trabajar del computador el Puerto de Comunicación 1 COM1 o el puerto de comunicación 2 COM2, y a su vez se puede programar el protocolo de comunicación dentro del cual esta:
Velocidad:
Velocidad de transmisión de datos del computador a la impresora.
Bits:
Cantidad bits por palabra.
Paridad:
Método para identificar que el dato recibido es igual al trasmitido.
Bits de parada:
Para identificar el inicio y el termino de una palabra.
