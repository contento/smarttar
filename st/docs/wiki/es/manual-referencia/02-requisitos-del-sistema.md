# Requisitos del Sistema
| Componente | Requisito |
|----|----|
| Sistema operativo | MS-DOS 5.0 o posterior |
| Procesador | Intel 80286 o compatible |
| Memoria convencional | 640 KB (requerido por el cargador del extensor de DOS) |
| Memoria extendida | Administrada por el extensor de DOS Pharlap 286 |
| Pantalla | Adaptador gráfico EGA o VGA |
| Entrada | Teclado; ratón opcional |
| Impresora | Serial (COM) o paralela (LPT); 18–80 columnas |
| Interfaz hardware telefónica | Placa cluster STB-x (1–4 clusters de 8 cabinas cada uno) |
| Protección de copia | Llave EEPROM en puerto paralelo (versiones de producción) |
| Módem (opcional) | Compatible Hayes; cualquier velocidad soportada por el puerto COM |
