# System Requirements
| Component | Requirement |
|----|----|
| Operating System | MS-DOS 5.0 or later |
| Processor | Intel 80286 or compatible |
| Conventional memory | 640 KB (required by DOS extender loader) |
| Extended memory | Managed by Pharlap 286 DOS extender |
| Display | EGA or VGA graphics adapter |
| Input | Keyboard; mouse optional |
| Printer | Serial (COM) or parallel (LPT); 18–80 column |
| Hardware telephone interface | STB-x cluster board (1–4 clusters of 8 booths each) |
| Copy protection | Parallel-port EEPROM dongle (production builds) |
| Modem (optional) | Hayes-compatible; any baud rate supported by COM port |
